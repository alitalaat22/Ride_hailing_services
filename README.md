Ride Smarter - Carpooling & Bus Booking Platform

A modern, high-performance web platform designed to revolutionize urban mobility through carpooling and bus booking services. This project focuses on sustainability, cost-efficiency, and user safety.

🚀 Key Features

Smart Carpooling: Connect with drivers and passengers heading the same way.

Bus Ticketing: Search and book trips across more than 300 cities.

Dynamic Search: Real-time filtering for dates, locations, and pricing.

User Trust System: Verified profiles and transparent review systems.

Responsive Design: Fully optimized for mobile and desktop viewing.

🛠️ Tech Stack

Design: Figma (Professional UI/UX Kit)

Frontend: HTML5, CSS3 (Tailwind CSS), JavaScript

Version Control: Git & GitHub

📂 Project Structure

assets/: Contains images, icons, and logos.

src/: Main source code files.

docs/: Technical documentation and design assets.
